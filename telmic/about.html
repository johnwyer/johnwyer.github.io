「目の前に迫る巨大ステージ」「番組を彩る華やかなセット」「歌とシンクロするCG」
「一瞬で床が落とし穴に変わる演出」

 

TVのバラエティー番組やクイズ番組、コンサートなどでみかけるこのようなシーン。

これらのシーンの至る所に私たちテルミックの技術が隠されています。

せりあがるステージには特殊な機械装置、キラキラ光る番組のセットには様々な電飾やCG映像、複雑な仕掛けにはオリジナルの制御システムが施されています。

私たちテルミックは、「機械装置」「電飾」「映像」「制御システム」という４つの異なる技術を組み合わせて、数々の演出プランを形にしてきました。

ジャンルにとらわれないオリジナルの技術こそが私たちの強みです。

その一瞬の驚き・笑い・感動のために、独自の技術をフルに使って「世界に二つとないものを」創り出してまいります。

テルミックができること

Planning / Sales Departmentプランニング / 営業部

お客様のイメージをどうすれば形にできるのか。営業スタッフが窓口となり、ご要望に沿ったアイデアと技術をご提案いたします。

Machine Manufacturing / Manufacturing機械製造 / 製造部

どんなに巨大で大掛かりな機械装置でも、経験豊富なスタッフが自社工場にて設計・製造・塗装まで担当いたします。

Control system · CG / technology producing unit制御システム・CG / 技術製作部

発注された装置が演出通りに点滅し動き回るように、オリジナルの制御システムを開発いたします。また音楽にあわせたCG映像制作も承っております。

Setting · Operation / Division設営・オペレーション / 業務部

実際に現場へ赴き安全かつ丁寧に設営・撤去いたします。

演出プランに基づいて、見る人の心に残るようなオぺーレーションをご提供いたします。

Maintenance / Administration Department保守 / 管理部

自社の管理倉庫を構え、徹底した管理体制を行っています。

各種演出機材や造作物を保守点検し、トラブルを未然に防いでおります。

Product Development / Technology Development Department製品開発 / 技術開発部

お客様のニーズがどのように変化していくのか。世界の最先端にある演出機材を積極的に導入し、新たな製品を開発してまいります。

STEP1クライアント様からのご依頼

TV番組やコンサートの美術会社様、イベント制作会社様が私たちの主なクライアントです。

「円形のステージを作って回転させる演出をしたいんだ。」

お客様の一言から私たちのプロジェクトが始まります。

まずはお客様のイメージをお聞かせください。

STEP2受注・製造・製作

機械装置の大きさ、演出プラン、スケジュールなど要件を1つずつ確認しながらプランを作成します。

出来上がったプランに沿って機械装置の製造・制御システムの開発に入ります。いくつものパーツにわかれるような規模の大きな装置は、一度工場で仮組することで、設営時間の短縮や安全性の向上につなげています。

STEP3設営

ステージの設営状況を舞台監督や大道具・照明などの他のセクションと相談しながら、いよいよ作業スタートです。

まずは機械装置の搬入から始まります。その後、会場の図面をもとに正確に位置を出し、パーツを組んでいきます。組み上がり後は、演出プラン通りの動きをするか制御システムを確認し微調整に入ります。

STEP4本番・撤去

どのタイミングでどのように装置を動かすのか、舞台監督や演出家と念入りに確認します。

本番では、絶対にタイミングを外せない緊張感がありますが、落ち着いて正確に操作しなければなりません。本番終了後は興奮もさながら、撤去作業に入ります。短時間で機械装置を解体し、トラックへ積み込み、後日倉庫に戻って機材を整理します。
